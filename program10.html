Cookie

A delicious confection that is quite popular when it comes to sweets. Cookies come in various varieties including chocolate chip, raisin, and macademia nut. A very different type of cookie is a small text file stored on a user's computer by a web server.

A sample Java program: 
class DataClass{
public static void main(String[] args){
System.out.println("Here is some text.");
}
}

Using the pre element you can display text however you want with all the spaces and line breaks preserved.
